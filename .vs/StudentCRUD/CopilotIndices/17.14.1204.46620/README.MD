# Student Management System (StudentCRUD)

## Features
- CRUD operations for Students
- JWT Authentication
- SQL Server Database
- Layered Architecture

## Tech Stack
- .NET Core Web API
- Entity Framework Core
- SQL Server

## Setup Steps
1. Clone repository
2. Open in Visual Studio
3. Update connection string
4. Run Update-Database
5. Run project

## API Endpoints
- GET /api/students
- POST /api/students
- PUT /api/students/{id}
- DELETE /api/students/{id}
